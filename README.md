# oryxtuiviewer
View Russia's equipment losses (approved by Oryx) in Terminal

Scrape https://www.oryxspioenkop.com/ with [goquery](https://github.com/PuerkitoBio/goquery)
Vizualize data with [termui](https://github.com/gizak/termui)
